# Python Calculator

A very small and simple calculator written in Python. No external dependencies or pip packages required — just the Python interpreter.

## Features
- Basic arithmetic operations (add, subtract, multiply, divide)
- Command-line usage via `Main.py`

## Usage
Run the calculator with your system Python interpreter:

```bash
python Main.py
```

No installation is required. Just ensure you have Python installed and run the file.
